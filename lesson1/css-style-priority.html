4. 优先级顺序（从高到低）：

  1. !important规则
  2. 内联样式（style属性）
  3. ID选择器（#id）
  4. 类选择器（.class）
  5. 元素选择器（p, div等）
  6. 通用选择器（*）
  7. 继承的样式

5. 特异性计算规则：

  • 内联样式：1000分
  • 每个ID选择器：100分
  • 每个类选择器、属性选择器或伪类：10分
  • 每个元素选择器或伪元素：1分
