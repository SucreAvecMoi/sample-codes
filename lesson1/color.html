CSS颜色属性与设值方式

常见的需要设置颜色的CSS属性：

  • color - 文字颜色
  • background-color - 背景颜色
  • border-color - 边框颜色
  • outline-color - 轮廓颜色
  • box-shadow - 阴影颜色
  • text-shadow - 文字阴影颜色

颜色设值方式：

1. 颜色关键字

示例：
color: red;
background-color: lightblue;
border: 2px solid green;

2. 十六进制表示法

示例：
color: #FF0000;
background-color: #87CEEB;
border: 2px solid #008000;

3. RGB表示法

示例：
color: rgb(255, 0, 0);
background-color: rgb(135, 206, 235);
border: 2px solid rgb(0, 128, 0);

4. RGBA表示法（带透明度）

示例：
color: rgba(255, 0, 0, 1);
background-color: rgba(135, 206, 235, 0.5);
border: 2px solid rgba(0, 128, 0, 0.8);

5. HSL表示法（色相、饱和度、亮度）

示例：
color: hsl(0, 100%, 50%);
background-color: hsl(197, 71%, 73%);
border: 2px solid hsl(120, 100%, 25%);

6. 渐变色

示例：
background: linear-gradient(90deg, #ff7e5f, #feb47b);
示例：
background: radial-gradient(circle, #43cea2, #185a9d);
