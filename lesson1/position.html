CSS Position定位属性

1. Static定位（默认值）

- 元素按照正常文档流进行定位

- 不受top、right、bottom、left属性影响

Static定位

2. Relative定位

- 相对于元素原来的位置进行定位

- 不脱离文档流

Relative定位

3. Absolute定位

- 相对于最近的非static定位祖先元素进行定位

- 脱离文档流

Absolute定位

4. Fixed定位

- 相对于浏览器视口进行定位

- 脱离文档流

Fixed定位

5. Sticky定位

- 基于用户的滚动位置进行定位

- 在特定阈值前为relative，之后为fixed
