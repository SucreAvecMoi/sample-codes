JavaScript表达式

1. 算术表达式


let a = 5 + 3;     // 加法
let b = 10 - 4;    // 减法
let c = 3 * 4;     // 乘法
let d = 15 / 3;    // 除法
let e = 10 % 3;    // 取余

2. 比较表达式


let x = 5;
let y = "5";
console.log(x == y);   // 相等（值）
console.log(x === y);  // 严格相等（值和类型）
console.log(x != y);   // 不相等
console.log(x !== y);  // 严格不相等
演示效果：

3. 逻辑表达式


let p = true;
let q = false;
console.log(p && q);  // 与
console.log(p || q);  // 或
console.log(!p);      // 非

5. 条件（三元）表达式


let age = 20;
let status = age >= 18 ? "成年" : "未成年";

注意事项：

  • 表达式是由操作数和运算符组成的
  • 表达式总是会产生一个值
  • 运算符优先级需要特别注意
  • 使用括号可以明确运算优先级
  • 在进行比较时，建议使用严格相等(===)和严格不相等(!==)
