在HTML中使用JavaScript的两种方式

1. 直接在HTML中使用script标签

<script>
    // JavaScript代码直接写在这里
    document.write("这是通过内部JavaScript输出的文本");
</script>
演示效果：

2. 引用外部JavaScript文件

<script src="external.js"></script>

两种方式的比较：

  • 内部JavaScript（使用script标签）：
    • 适合简单的页面脚本
    • 方便快速修改和调试
    • 不需要额外的文件请求
  • 外部JavaScript文件：
    • 代码可以被多个页面复用
    • 便于维护和版本控制
    • 浏览器可以缓存文件，提高加载速度
    • 可以并行下载，提高页面加载效率

最佳实践：

  • 将script标签放在body结束标签之前，可以提高页面加载速度
  • 对于大型项目，推荐使用外部JavaScript文件
  • 可以使用async或defer属性优化脚本加载
